Datastuctures:
    list : kind of arr like C/C++ 
         : dynamica array
         - read / write collection 
            - change the element of collection 
            - add / remove the elements of collection 
         : collection of data 
         : use it as collection or to represent composite data like row 

         employees = [ ]    # empty list 
         employees = list() # empty list  

    tuple: similar to list 
         - readonly collection
            - cant change the element of collection 
            - cant add / remove the elements of collection 

         student = ('Dravid', 52, 450000, True)

    dictionary:    
        - collection of key value pairs 
            - key must be primitive type or tuple 
                str, int, float, bool, tuple 
        - read / write collection 
            - change the element of collection 
                value of key-value 
            - add / remove the elements of collection 
                ie key-value pair 
        - to represent the human understandable composite site 

        jaiswal = {'name': 'Jaiswal', 'age':23, 'salary':300000, 'is_active':True}

        freq = {} # empty dict 
        freq = dict() # empty dict
