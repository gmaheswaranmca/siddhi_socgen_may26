__str__() and __repr__() are 
    both special methods used 
    to convert an object into a string, 
    but Python uses them in different situations.

__str__() Used for a readable/user-friendly display.
    Called by: print(obj), str(obj)

__repr__()
    Used for a developer/debug representation. 
    Called by: repr(obj)
    Also used when object is inside collections:
    print([obj])
